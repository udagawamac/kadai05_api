## ②課題内容（どんな作品か）
- Bingmap APIと連携させたアプリ
- BingMaps GO!のサンプルを利用
- 地図上でクイズに答え、アイテムとキーワードをゲットしていく

https://udagawamac.github.io/kadai05_chat/
